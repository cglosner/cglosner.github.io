## Kernel CVE Analysis

A collaborative research effort at Purdue University ([purseclab/kernelcveanalysis](https://github.com/purseclab/kernelcveanalysis)) focusing on Linux and Android kernel exploitation research.

### Components
- **Kexploit** — Framework for automatically adapting kernel exploit PoCs across different kernel versions
- **Crash Analyzer** — Tool for reproducing syzbot-detected crashes in target Android environments
- **Android Environment Analyzer** — Toolkit for auditing runtime environments and mapping access-control boundaries
- **Vulnerability Knowledge Base (VKB)** — Database of kernel vulnerabilities with associated PoC source code

### Technologies
- Java, C, Python, Syzkaller, Android, Linux Kernel, QEMU
